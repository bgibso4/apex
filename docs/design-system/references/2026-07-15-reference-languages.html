Same screen, four languages

Round 1 is dead — too many hues, too saturated, swatch walls that shouted. Every frame below obeys the corrected principles: one tint color (iOS's single-accent model), 60-30-10 (neutral-dominant, accent ≤10% and only as information), desaturated-on-dark (muted tones that don't glow), and color = meaning, never decoration. The four block colors are gone in all of them. What differs is personality — pick the one (or the blend) that feels like APEX.
