Gradient treatments — round 3

Four ways a gradient can live inside the locked principles (muted, neutral-dominant, color = meaning). The rule that keeps gradients tasteful: they are atmosphere, not decoration — nearby hues only, whisper-low contrast, never competing with content. Ordered subtle → noticeable. All four are trivial in SwiftUI (LinearGradient / RadialGradient), and can breathe — e.g. the glow warming slightly when a session is live.
